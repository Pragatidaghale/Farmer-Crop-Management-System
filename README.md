# 🚜 Farmer Crop Management & Subsidy System

## 📌 Project Overview
Salesforce-based agriculture management system for subsidy processing.

## 🔥 Features
- Farmer crop registration
- Automatic subsidy approvalrejection
- Validation rules
- Reports & dashboards
- Crop status tracking

## 🧱 Custom Object
Farmer_Crop__c

## ⚙️ Technologies Used
- Salesforce Admin
- Apex Trigger
- Apex Class
- Validation Rules
- Reports & Dashboards

## ⚡ Apex Logic
Subsidy eligibility automatically calculated using
- Income
- Land size

## 📊 Reports
- Approved Farmers
- Rejected Farmers
- Crop Distribution

## 🎯 Interview Explanation


This project is developed using Salesforce Admin and Apex. 
The system helps manage farmer crop details and automatically decides subsidy eligibility based on farmer income and land size. 
Admin users can verify applications, update status, and monitor reports and dashboards.
